### 🌱 [Plan4Impact](https://github.com/lieuryan/Plan4Impact)
An **AI-powered sustainable event planning platform** that helps businesses organise eco‑friendly events while tracking **carbon footprint** and **budget** in real time.

**Highlights:**
- 💬 *AI Chatbot* assists with event setup through natural conversation.  
- 📊 *Impact Dashboard* displays live carbon and budget metrics.  
- 🏢 *Venue & Service Selection* with verified sustainability ratings.  
- 🌍 *Carbon Offset Integration* connects users with real-world Australian offset projects.  
- 🎨 *Modern Interface* built with **React 18 + TypeScript, Vite, Framer Motion**, and an eco‑friendly theme.

**Tech Stack:** React, TypeScript, CSS3, Framer Motion, n8n webhook.
